# numberAdditionGame

A quick game on numbers and **adding** them

## Requirements
- Number generated [17-29 inclusive]
- User shown a set of numbers from a roll die
  - Must decide which numbers to add to try and get as close to the number as possible
  - Must be at least 5 away from the target for it to be valid

Functions for:
- dice role
  - using random lib
- random number
  - using random lib
- Additions
  - takes inputs if additions are wanted or not
  - while loops needed for repeated additions
